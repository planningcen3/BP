# วิธีเชื่อมเว็บ PEA Budget กับ Google Sheet

ไฟล์สคริปต์ในโฟลเดอร์นี้มี 2 ชุด:

- `Code.gs` สำหรับ `งบทำการ`
- `Code-kpj2.gs` สำหรับ `งบ คพจ.2`

## ขั้นตอนติดตั้ง

1. สร้าง Google Sheet ใหม่ หรือเปิดไฟล์ Sheet ที่ต้องการใช้เก็บข้อมูล
2. ไปที่ `Extensions > Apps Script`
3. ลบโค้ดเดิมในโปรเจกต์
4. เลือกไฟล์สคริปต์ให้ตรงกับชุดงบที่ต้องใช้
5. วางโค้ดลงใน Apps Script
6. แก้ค่า `SPREADSHEET_ID` ให้เป็น Google Sheet เป้าหมายของงบนั้น
7. กด Save
8. กด `Deploy > New deployment`
9. เลือกชนิดเป็น `Web app`
10. ตั้งค่า:
   - Execute as: `Me`
   - Who has access: เลือกตามนโยบายหน่วยงาน เช่น `Anyone with the link` หรือภายในองค์กร
11. กด Deploy แล้วอนุญาตสิทธิ์ Google Sheet
12. Copy URL ที่ลงท้ายด้วย `/exec`
13. กลับมาที่เว็บ PEA Budget แล้วเลือกชุดงบให้ตรง จากนั้นวาง URL ในช่อง `Apps Script Web App URL`

## โครงสร้างชีตของ `งบทำการ`

ใช้ไฟล์ `Code.gs`

- `BudgetConfig` สำหรับหมวดงบและวงเงินตั้งต้น
- ชีตตามหมวดจริง เช่น `ล่วงเวลา`, `ค่าที่พัก`, `ค่าเบี้ยเลี้ยง`

รูปแบบแต่ละชีตหมวด:

- `ที่`
- `ชื่อ - สกุล`
- `เบิกแล้ว`
- `คงเหลือ`
- `เบิกครั้งนี้`
- `ยอดคงเหลือ`
- `วัน/เดือน/ปี`
- `รายละเอียด/เลขที่เอกสาร`
- `หมายเหตุ`

## โครงสร้างชีตของ `งบ คพจ.2`

ใช้ไฟล์ `Code-kpj2.gs`

- `BudgetConfig` สำหรับหมวดงบและวงเงินตั้งต้น
- ชีตตามหมวดจริง เช่น `ค่าเบี้ยเลี้ยง`, `ค่าที่พัก`, `ค่าล่วงเวลา`, `ค่าเดินทาง`

รูปแบบแต่ละชีตหมวด:

- `ที่`
- `ชื่อ - สกุล`
- `เบิกแล้ว`
- `คงเหลือ`
- `เบิกครั้งนี้`
- `ยอดรวม`
- `วัน/เดือน/ปี`

ถ้ากด `อัปโหลดข้อมูลเดิม` จากหน้าเว็บ ระบบจะสร้าง `BudgetConfig` ก่อน แล้วค่อยแยกข้อมูลลงแต่ละชีตหมวดตามรายการเดิมของงบนั้น

หลังแก้โค้ด Apps Script ต้องไปที่ `Deploy > Manage deployments` แล้วกดแก้ไข deployment เพื่ออัปเดตเวอร์ชันที่ใช้งานจริง
